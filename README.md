# Frontend - Aplikasi Booking Ruangan Meeting

Project ini menggunakan **React + Vite**, **Tailwind CSS**, dan **Axios** untuk komunikasi API ke backend.

## 🚀 Fitur
- Form Booking Ruang Meeting
- Select Ruangan & Auto Kapasitas
- Kalkulasi Konsumsi
- Validasi Form
- Responsive Layout

## 🛠️ Teknologi
- React (Hooks)
- React Router
- Tailwind CSS
- Axios

## 📦 Instalasi

```bash
cd frontend
npm install
